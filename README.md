# 🐍 AI Snake Game

A classic Snake game where you can either play manually **or** watch an AI learn to play using Q-Learning (Reinforcement Learning).

## Features
- Manual mode (arrow keys)
- AI mode (self-learning agent)
- Real-time rendering with Pygame
- The AI improves over episodes

## Install & Run
```bash
pip install pygame numpy
python ai_snake_game.py
